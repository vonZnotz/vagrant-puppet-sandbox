typo3base vagrant
=================

## Description

This is a simple lamp stack sandbox

## Installation

Make use of librarian-puppet to get avoid of git submodules

https://github.com/voxpupuli/librarian-puppet

	cd environments/dev
	librarian-puppet install

You're done!

## Make it run

	vagrant up
